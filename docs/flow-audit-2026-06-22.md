# moTF 1단계 전체 플로우 점검 결과

점검일: 2026-06-22

## 통과

- 이용자 회원가입, 이메일 인증, 로그인, 마이페이지 진입
- DB 원본 객실 가격을 사용한 실제 예약 요청
- 로그아웃·재로그인 후 예약 기록 유지
- 이용자와 업장 간 실제 채팅 저장 및 재조회
- 이용자 문의 접수와 처리 상태 조회
- 관리자의 전체 예약 조회
- 관리자의 전체 채팅 읽기 전용 모니터링
- 관리자의 문의 상태 변경과 이용자 화면 반영
- 비로그인 사용자의 회원·예약·주문·채팅 테이블 접근 차단
- 승인된 업장과 활성 상품만 공개 조회
- 미완성 결제 API의 거래 생성 차단

## 점검 중 발견해 수정한 내용

- 이메일 미인증 로그인 오류를 구체적으로 안내
- 브라우저 입력 가격 대신 DB 가격으로 예약·주문 계산
- 잘못 저장된 파트너 역할과 승인 상태 복구 SQL 추가
- 권한 검사 전에 이전 대시보드가 표시되던 로그인 순서 수정
- 신규 파트너의 샘플 업장·객실 데이터 제거
- 신규 파트너 로그인 시 업장 등록 화면 자동 이동
- 상품 수정 시 기존 상품 ID 유지

## 보류

- 신규 파트너 가입·관리자 승인·로그인 재시험
- 신규 숙소 기본정보·객실·사진 저장
- 사장님의 예약 확정·거절과 채팅 답변
- 공판장 등록, 상품 등록, 주문 생성과 처리

보류 사유: 테스트에 사용한 신규 파트너 Auth 계정이 삭제되어 당분간 재생성이 어려움.
